# TalentFilter AI Frontend

`cp .env.example .env`, ardından `npm install` ve `npm run dev` ile başlatılır.
Backend varsayılan olarak `http://localhost:8000/api/v1` adresinde çalışmalıdır.

Kontroller: `npm run lint`, `npm run format:check`, `npm run typecheck`, `npm run test`, `npm run build`.

MVP; dashboard/ilan/ad ay/match/shortlist akışlarının temel ekranlarını içerir. Query ve sonuç import ekranları bu teslimatta yer tutucu durumundadır.
